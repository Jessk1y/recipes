# 🍳 Мои рецепты

Личная коллекция рецептов: ингредиенты и способ приготовления.
Статический сайт, работает на GitHub Pages — без серверов и оплаты.

## Как добавить рецепт

Все рецепты лежат в одном файле: [`data/recipes.json`](data/recipes.json).
Чтобы добавить новый — допиши объект в массив:

```json
{
  "id": "unikalnoe-imya",
  "title": "Название блюда",
  "category": "Завтраки",
  "tags": ["завтрак", "быстро"],
  "image": "images/foto.jpg",
  "time": "30 мин",
  "servings": "2 порции",
  "ingredients": [
    "Ингредиент 1 — 100 г",
    "Ингредиент 2 — 2 шт"
  ],
  "steps": [
    "Шаг первый.",
    "Шаг второй."
  ]
}
```

### Поля
| Поле | Обязательное | Описание |
|------|--------------|----------|
| `id` | да | Уникальный латиницей, без пробелов (используется в ссылке) |
| `title` | да | Название рецепта |
| `category` | нет | Одна категория (Завтраки, Супы, Десерты…) |
| `tags` | нет | Список тегов для поиска и фильтра |
| `image` | нет | Путь к фото (`images/файл.jpg`). Если пусто — покажется эмодзи |
| `time` | нет | Время приготовления |
| `servings` | нет | Количество порций |
| `ingredients` | да | Список строк |
| `steps` | да | Список шагов |

## Фото
Кидай фото блюд в папку [`images/`](images/) и указывай путь в поле `image`.

## Локальный просмотр
Открой `index.html` через локальный сервер (просто двойной клик не подойдёт из-за `fetch`):

```bash
python -m http.server 8000
# затем открой http://localhost:8000
```
